# IZAN Viral Scripter

Analyzes a viral reference video and rewrites it using the IZAN Viral Scripting
Framework — keeping the exact same hook and overall structure, replacing only
the value with the creator's own expertise. For text-overlay reels (no speech),
the caption becomes the source material to adapt.

---

## First-Time Setup

On first run, ask the user for:

1. **Knowledge Base** — Either:
   - A Notion page URL they want to load
   - Or a direct description of their expertise, results, tools, and speaking style

2. **Reference video** — Either:
   - A transcript they paste directly
   - Or an Instagram/TikTok/YouTube URL they want analyzed

Store for the session.

## THE IZAN VIRAL SCRIPTING FRAMEWORK

### Core Philosophy
1. **Clarity** — Simple language, clear structure, specific examples
2. **Conversion** — Strategic CTAs, value delivery, engagement triggers
3. **Curiosity** — Pattern interrupts, open loops, psychological triggers

---

### Critical: Video Length

**Every video must be as SHORT as possible while delivering all value. Non-negotiable.**

- If a 60-second video can deliver the same value in 35 seconds, it MUST be 35 seconds
- Attention spans are decreasing — every second must earn its place
- Algorithms favor completion rates; shorter videos have higher share rates
- Always aim 10-20% shorter than the reference if possible without sacrificing value

**Length targets by content type:**
| Type | Target |
|------|--------|
| Demonstration | 20-40 sec |
| Tutorial | 30-60 sec |
| Educational | 45-60 sec |
| Story-based | 45-60 sec |

---

### The 5-Part Structure

---

#### 1. HOOK (0-5 seconds)

Must immediately stop the scroll.

**Requirements:**
- Under 10 words
- Creates clear stakes
- Generates immediate curiosity
- Personal impact

**Hook Types:**
- **Problem-Based:** "You're losing X because of Y"
- **Pattern Interrupt:** "Stop doing X until you see this"
- **Counter-Intuitive:** "This X is actually destroying your Y"
- **Numbers-Based:** "These 3 things are killing your X"

---

#### 2. SELLING THE SOLUTION (5-15 seconds)

Builds anticipation WITHOUT revealing the solution yet. Creates curiosity, builds
credibility, promises transformation.

**Section flexibility — this is critical to get right:**
- **Demonstration videos:** Skip this section entirely. The demo IS the interest peak.
- **Tutorial videos:** Compress to 3-5 seconds
- **Educational content:** Full 10 seconds
- **Story-based:** Slightly expanded

**The 7 Interest Peak Types:**

**1. RISK REVERSAL** — Reassures the viewer it's worth their time
*"And the crazy thing is this will literally take under a minute to set up"*

**2. AUTHORITY ENDORSEMENT** — Borrows credibility from a known authority
*"This is the same technique [respected name in niche] used to [result]"*

**3. CONTROVERSIAL** — Validates or challenges the viewer's opinion
*"And yet 99% of the people watching won't make it to the end of this"*

**4. PERSONAL STORY** — Uses the creator's own results as proof
*"This is the exact thing I did to [specific, concrete result]"*

**5. NEGATIVE ASSUMPTION** — Addresses objections before they arise
*"And no, it's not [the obvious thing]. It's something most people completely overlook"*

**6. HYPE UP** — Builds anticipation and excitement
*"I was going to keep this to myself — but I'm giving it to you for free"*

**7. CALL OUT** — Highlights a mistake the viewer is likely making
*"This is exactly where most people screw it up"*

---

#### 3. GIVING THE PRINCIPLE (15-45 seconds)

Deliver the promised value clearly and actionably.

- Clear steps or structured points
- 5th-grade language — short sentences, plain words
- Specific examples
- Show the transformation
- Keep each point tight

---

#### 4. MAKING IT APPLICABLE (45-55 seconds)

Show exactly how to implement the solution.

- Immediately actionable
- Specific next step
- Clear expected result

Openers: *"Here's what to do..."* / *"Try this right now..."* / *"Start today by..."*

---

#### 5. CALL TO ACTION (55-60 seconds)

**Format:** *"I've created [specific resource] with [clear benefit]. Comment [KEYWORD] and I'll send it to you"*

- Specific resource (not vague "more content")
- Keyword trigger (comment-to-DM automation)
- Clear benefit
- Urgency or reason to act now

---

### Content Type Patterns

| Type | Selling Solution | Why |
|------|-----------------|-----|
| Demonstration | Skip entirely | The demo itself creates the interest peak |
| Tutorial | 2-5 seconds | Value is in the steps, not the setup |
| Educational | Full 10 seconds | Needs credibility before delivering insights |
| Transformation | Slightly expanded | Before/after contrast creates the value |

## Phase 1 — Load Creator Profile

From the knowledge base or user description, extract:
- Voice and tone (how they naturally speak)
- Signature frameworks or systems (names and core ideas)
- Social proof (specific results, numbers, client outcomes)
- Tools or methods they reference
- Recurring language patterns

## Phase 2 — Analyze the Reference Video

Break down the transcript section by section:

```
REFERENCE VIDEO ANALYSIS
Total length: ~X seconds
Content type: [Demonstration / Tutorial / Educational / Story-based / Transformation]

HOOK (0-Xs): [type] — "[exact hook text]" — [word count]
SELLING SOLUTION (X-Xs): [Interest Peak type] OR [SKIPPED — and why it works without it]
PRINCIPLE (X-Xs): [structure and summary of value delivered]
APPLICATION (X-Xs): [how they make it actionable]
CTA (X-Xs): [resource offered] / [keyword] / [benefit promised]

Key observations:
- [What makes this video perform]
- [Which sections are compressed/skipped and why]
```

## Handling Text-Overlay Reels (No Speech)

Many viral reels — especially short ones under 15 seconds — have no spoken words at all.
They use text on screen with background music. When you encounter one of these:

**Detect it:** The transcript will be empty, a single word, or just background music
notation. The video duration is usually under 15 seconds.

**Where to find the script:** The real content lives in the **caption**. If the reel
delivers value, it does it through the caption text. Use the caption as your source
material — that's the script you're adapting, not the (empty) transcript.

**How to rewrite:**
- Keep the same text-on-screen format — your version should also be a text-overlay reel
- Keep the same text that appears on screen (the hook line / title text) verbatim,
  just like you'd keep a spoken hook verbatim
- Adapt the caption content with your own value, stories, and proof points
- Match the same emotional tone and length of the original caption
- If the original uses a list of confessions/thoughts/tips in the caption, keep that
  same list format but swap in your own version

**Output format for text-overlay reels:**

```
REWRITTEN SCRIPT (Text-Overlay Format)
Estimated length: ~X seconds

[ON-SCREEN TEXT]
"[same hook/title text from original]"

[CAPTION]
[Adapted caption using creator's value, voice, and proof points —
same structure and emotional tone as the original caption]
```

## Phase 4 — Rewrite the Script

**Rewrite rules:**
1. **Keep the exact same hook verbatim.** The hook already went viral in your niche — don't reinvent it. Copy it word for word. The hook is proven; changing it only adds risk.
2. **Keep the overall structure and flow of the original video.** The sequence of sections, the pacing, which parts are compressed or expanded — mirror all of it. The original creator already found the rhythm that works. You're swapping the value inside, not rebuilding the frame.
3. Match the total length of the reference video
4. Creator's knowledge, proof points, and tools replace the original value sections
5. Creator's voice throughout (from their profile)
6. 5th-grade language — plain words, short sentences
7. Must sound like a real person talking, not written text

**Output format:**

```
REWRITTEN SCRIPT
Estimated length: ~X seconds (~X words)

[HOOK — 0 to Xs]
"..."

[SELLING THE SOLUTION — Xs to Xs]
"..."
← or: [SKIPPED — demonstration format]

[PRINCIPLE — Xs to Xs]
"..."

[APPLICATION — Xs to Xs]
"..."

[CTA — Xs to Xs]
"..."

---
KEPT: exact hook / overall structure and flow / [which sections / pacing]
CHANGED: value → [what was swapped in] / CTA → [keyword]
```

## Phase 5 — Save to Notion (if applicable)

If this script is linked to a content idea in Notion, update the page:

```
notion-update-page(
  command: "update_content",
  page_id: [content idea page ID],
  content_updates: [{
    old_str: "[existing section marker]",
    new_str: "## Rewritten Script\n\n[script with section labels]\n\n[existing section marker]"
  }]
)
```

Update status to "SCRIPT TO APPROVE":
```
notion-update-page(
  command: "update_properties",
  page_id: [content idea page ID],
  properties: { [status field]: "SCRIPT TO APPROVE" }
)
```
